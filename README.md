## どんなツールか

**瓦版工房 PRO — 社内記事ジェネレーター v4.0**

項目を埋めると、AIに渡すプロンプトまたは穴埋め式の下書きが出てくるツールです。「書く内容は頭の中にある、でも形にならない」という場面に使えます。

| | |
|---|---|
| 動作 | HTMLファイルをブラウザで開くだけ |
| 推奨ブラウザ | **Chrome** |
| インストール | 不要 |
| データ | 端末内のみ保存・外部送信なし |
| オフライン | 可（AIへのプロンプト貼り付けは別途必要） |

---

## 5タブでできること

### 📋 企画を立てる

記事を書く前の「整理」タブです。誰の・何の課題を解決する記事か、ペルソナ（名前・レベル・状況・なりたい姿）、Why／What／Howを入力して企画シートを刷れます。「記事フォームへ反映」を押せばそのまま執筆へ運べます。

### 🖋 プロンプトを作る

入力内容をChatGPT／Claudeに貼れる指示文に変換します。記事の型は2種類から選べます。

**八章立て（しっかり）**  
記事の標準構成。以下の項目を入力します。

- 記事タイトル・サブタイトル・作成者
- 立場・場面 / 適用範囲
- 対象読者（1行1人物像）
- 避けるべき点（1行1項目／補足は「項目／補足」形式）
- 基本方針の一文
- フロー（分岐・ループをテキストで）
- 「○○集」の名称・件数・グループ観点（A/B/C）
- 比較する立場A・B と比較軸
- 締めのメッセージ
- オプション：Word化の指示 / フロー図PNGの指示 / 社外公開版の指示

**簡易の覚書（さっと）**  
手順メモ・補足ページ向けの軽い構成。タイトル・要約・前提・まとめ・想定読者を入力します。オプションでNotion貼り付け版の指示も付けられます。

### ✍️ 記事を書く

「記事を作って穴埋めへ」で下地を生成し、**［ 記入 ］** の空欄を埋めていく形式です。Enterで次の空欄へ移動。文字数と読了時間も表示されます。整形プレビューで仕上がりを確認できます。

### 📥 ひな形ダウンロード

選んだ型の空欄テンプレを `.md` でダウンロードできます。Notionや社内エディタに貼り付けて、ツールなしで直接書きたい場合に。

### 📖 使い方（高札）

各タブの手引きをまとめたヘルプページです。

## 出力・保存

- **写し取る（コピー）** — クリップボードに全文コピー
- **Wordで保存（.doc）** — Wordファイルとして書き出し
- **PDFで保存（印刷）** — ブラウザ印刷でPDF化
- **文箱（ふばこ）** — 端末に名前をつけて一時保存。タブを閉じても自動で控えます
- **下書きエクスポート（.json / CSV）** — 別端末や日を改めて続きを再開
